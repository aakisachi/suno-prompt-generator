# Sunoプロンプト生成

複数のLyricsをSunoのCustomモードへ順番に入力し、Createを押すChrome拡張機能です。

> [!IMPORTANT]
> 非公式ツールです。Suno, Inc.とは関係ありません。利用前にSunoの最新の利用規約を確認し、自分の責任で使用してください。

## 主な機能

- 複数のLyricsをキューへ追加
- CSV読込
- 順次生成、停止、再開
- ポップアップを閉じた状態でのバックグラウンド実行
- 失敗した項目の再試行
- キューの一括削除
- 生成間隔の設定

## インストール

### ZIPを使う場合

1. GitHubの「Code」から「Download ZIP」を選びます。
2. ZIPを展開します。
3. Chromeで `chrome://extensions/` を開きます。
4. 右上の「デベロッパー モード」をオンにします。
5. 「パッケージ化されていない拡張機能を読み込む」を押します。
6. 展開したフォルダ内の `dist` フォルダを選びます。

### Gitを使う場合

```bash
git clone https://github.com/aakisachi/suno-prompt-generator.git
```

クローンしたフォルダ内の `dist` をChromeへ読み込んでください。

## 使い方

1. Chromeで `https://suno.com/create` を開き、Customモードにします。
2. Chrome右上の拡張機能から「Sunoプロンプト生成」を開きます。
3. 1行に1件のLyricsを貼るか、CSVを読み込みます。
4. 「キューに追加」を押します。
5. 「開始」を押します。
6. 停止後に再開すると、未処理の曲から続けます。

開始後は拡張機能のポップアップを閉じても、別タブへ移動しても、Chromeを最小化しても処理が続きます。進捗は拡張機能アイコンのバッジ（数字表示）と、ポップアップ内のキューで確認できます。

CSVは `曲名,Lyrics` の2列です。既存の `曲名,プロンプト` 形式も読み込めます。

## Chrome権限

| 権限 | 用途 |
| --- | --- |
| `storage` | キュー、進捗、生成間隔をChrome内へ保存するため |
| `activeTab` | 拡張機能を開いたSunoタブへ操作命令を送るため |
| `alarms` | ポップアップを閉じた後も次の曲を処理するため |
| `suno.com` / `app.suno.ai` | Lyrics入力欄とCreateボタンを操作するため |

この拡張機能は、外部の独自サーバーへデータを送信しません。アクセス先はSunoの対象ページに限定しています。解析ツールや広告も入れていません。

## 注意事項

- Sunoの画面構成が変更されると、Lyrics入力欄やCreateボタンを見つけられなくなる場合があります。
- CAPTCHA（人間確認）やアクセス制限を回避しません。表示された場合は停止します。
- Chromeを完全終了した場合やMacがスリープした場合は処理できません。Chrome再起動時にSunoタブが復元されていれば、保存済みのキューから再開します。
- 処理対象のSunoタブを閉じると、安全のためキューを一時停止します。
- 生成間隔は設定ボタンから10〜300秒で変更できます。既定値は20秒です。
- キューの内容はChromeのローカルストレージ（端末内の保存領域）へ保存されます。

## 開発

```bash
npm install

npm run build
```

ビルド結果は `dist` に出力されます。
